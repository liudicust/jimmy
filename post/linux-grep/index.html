重学grep 命令

  1. 简单从文件中查找关键词
1

# grep root /etc/passwd
  1. 查找的时候忽略大小写
1

# grep -i root /etc/passwd
  1. 查找的时候忽略一些词
1

# grep -v root /etc/passwd
  1. 递归查找
1
2

grep -r -i root /etc
grep -R -i root /etc #会从软连接指向的目录继续查找
  1. 强制查找单词，而不是所有包含关键词的位置
1

grep -w -i root /etc
  1. 一次查找多个关键词
1
2

grep -E -w -i "average|memory" kubelet.log
grep -e mem -e [kK]B kubelet.log #查找匹配kB或者KB
  1. 查找的匹配的个数
1

grep -c -i "memory" kubelet.log
  1. 查找的时候带上行号
1

grep -n "memory" kubelet.log
  1. 检索的时候限制展示数量
1

grep -n -m 5 "memory" kubelet.log
  1. 检索的时候，展示关键词的上下文（上下行）
1
2
3

grep -A 3 -i "memory" kubelet.log #展示后面几行
grep -B 3 -i "memory" kubelet.log #展示前面面几行
grep -C 3 -i "memory" kubelet.log #展示前后几行
  1. 检索的时候展示匹配关键词所在的文件名
1
2

grep -l "stdio.h" *.c #包含关键词的文件
grep -L "stdio.h" *.c #不包含关键词的文件
  1. 查找开头或结尾包含关键词
1
2

grep "^ " kubelet.log #包含空格开头的
grep "error$" #包含以error为结尾的
  1. 和管道符号结合
1
2

grep "key" *.log | less 
grep "passwd" *.log |wc -l #统计行数
